``if`` ... ``elif`` ... ``else``
--------------------------------

.. code-block:: python

    if ... :
         print('this is true')
     else:
         print('this is false')


.. code-block:: python

    if not name == 'Agata':
         print('this is false')
     else:
         print('this is true')


.. code-block:: python

    if name != 'Agata':
         print('this is false')
     else:
         print('this is true')


.. code-block:: python

    if name == 'Agata':
        print('Your name is Agata')
    elif name == 'Borys':
        print('Your')
    else:
         print('Twoje imie to nie jest ani Borys ani Agata')
