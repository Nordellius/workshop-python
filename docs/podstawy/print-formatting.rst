****************
Print formatting
****************

Stary styl - ``%s``
===================

* kolejnościowe
* nazwane
* typy: ``string``, ``int``, ``float``
* operatory na stringu

Nowy styl - ``.format()``
=========================

* ``string``
* ``int``
* ``float``
* operatory na stringu
* jako parametry do ``print("string", **args)``

f-strings - Python >= 3.6
=========================

* ``f'{variable}'``
* ``f'{self.field}'``
* ``f'{datetime:%Y-%m-%d %H:%M}'``
